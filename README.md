# Dollar Monitor

A single-page dashboard that answers one question in five seconds: **"Is the dollar strong or weak, and why?"**

Built for macro-aware individuals — investors, analysts, founders — who want signal without noise. One score, one regime label, one sentence of AI-generated interpretation. Open it, read it, close it.

![Dollar Monitor Screenshot](screenshot.png)

## How it works

The worker fetches four data series from FRED, computes derived values, scores the dollar, classifies the current regime, and calls Claude to generate a one-sentence interpretation. The frontend displays the result. That's it.

```
FRED API (4 series) → Cloudflare Worker → Score + Regime + Interpretation → Static Frontend
```

### The score

**USD Score** is a weighted composite of three normalized signals:

| Weight | Signal | What it captures |
|--------|--------|-----------------|
| 40% | Real yield (z-scored) | Are dollar-denominated assets attractive after inflation? |
| 30% | DXY trend (MA20 vs MA100) | Is the dollar gaining momentum? |
| 30% | Inverted S&P 500 trend | Is money leaving risk assets for safety? |

Score > +0.20 = **STRONG USD**. Score < -0.20 = **WEAK USD**. In between = **NEUTRAL**.

### The regime

The regime label explains *why* the dollar is behaving the way it is:

- **RISK-OFF DOLLAR RALLY** — Real yields rising, stocks falling. Flight to safety.
- **YIELD-DRIVEN STRENGTH** — High real yields with DXY trending up. Rate differentials attracting capital.
- **STAGFLATION WATCH** — Real yields rising, stocks falling, *and* inflation expectations rising.
- **RISK-ON WEAKNESS** — Real yields falling, stocks rising. Capital leaving safe havens.
- **NEUTRAL / MIXED** — No dominant pattern.

### The interpretation

One sentence, generated by Claude (Haiku), explaining what's driving the dollar right now. It receives all raw and derived data and produces a concise, specific take — not a hedge.

## Stack

| Layer | Choice |
|-------|--------|
| Frontend | HTML + vanilla JS + Tailwind CDN |
| Backend | Cloudflare Worker |
| Data | FRED API (free, official) |
| Interpretation | Claude API (Haiku) |
| Deploy | Cloudflare Pages + Workers |

No framework. No build step. One `index.html`, one `app.js`, one worker.

## Setup

### 1. Deploy the worker

```bash
cd worker
npm install

# Set your API keys as secrets
wrangler secret put FRED_API_KEY
wrangler secret put ANTHROPIC_API_KEY

# Deploy
npm run deploy
```

### 2. Configure the frontend

Update `API_URL` in `frontend/app.js` to point to your deployed worker URL.

### 3. Serve the frontend

Any static host works — Cloudflare Pages, Netlify, Vercel, or just open `index.html` locally.

```bash
cd frontend
npx serve .
```

### Secrets required

| Secret | Source | Cost |
|--------|--------|------|
| `FRED_API_KEY` | [fred.stlouisfed.org](https://fred.stlouisfed.org/docs/api/api_key.html) | Free |
| `ANTHROPIC_API_KEY` | [console.anthropic.com](https://console.anthropic.com/) | ~$0.001/request |

## Data sources

All data from [FRED](https://fred.stlouisfed.org/) (Federal Reserve Bank of St. Louis), updated daily with a one-business-day lag:

| Series | Description |
|--------|------------|
| DTWEXBGS | Trade-Weighted U.S. Dollar Index |
| DGS10 | 10-Year Treasury Constant Maturity Rate |
| T10YIE | 10-Year Breakeven Inflation Rate |
| SP500 | S&P 500 Index |

## What this is not

- Not a trading terminal
- Not real-time (FRED data is delayed one business day)
- Not financial advice
